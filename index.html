grd

Welcome to grd — a flexbox-based layout system.

Similar to Bootstrap, the we create rows which contain one or more columns.

<div class="grd-row">
<div class="grd-col-4-12">
<div class="grd-col-8-12">

But why stop at 12-based grid... use any base!

<div class="grd-row">
<div class="grd-col-3-5">
<div class="grd-col-2-5">

Mix and match bases!

<div class="grd-row">
<div class="grd-col-1-4">
<div class="grd-col-2-8">
<div class="grd-col-1-2">

The widths don't have to add-up to 100%.

<div class="grd-row">
<div class="grd-col-1-3">
<div class="grd-col-1-3">
<div class="grd-row">
<div class="grd-col-1-3">
<div class="grd-col-1-3">
<div class="grd-col-1-3">
<div class="grd-col-1-3">

Perhaps we want columns to take up just as much width as their content need

<div class="grd-row">
<div class="grd-col-inline">
<div class="grd-col-inline">
<div class="grd-col-inline">

This is so useful, a shorter syntax exists:

<div class="grd-row">
<div>
<div>
<div>
<div class="grd-row"> <input type="text">
<div>

Some cells can dynamically take up the remaining width

<div class="grd-row">
<div class="grd-col-inline">
<div class="grd-col-inline">
<div class="grd-col-auto">
<div class="grd-row">
<div class="grd-col-2-6">
<div class="grd-col-auto">
<div class="grd-col-1-6">

Multiple grd-col-auto share the remaining width equallyAs long as they don't overflow

<div class="grd-row">
<div class="grd-col-2-6">
<div class="grd-col-auto">
<div class="grd-col-auto">
<div class="grd-col-1-6">

Maybe you want to pull the last columns to the endOnly the first pulled colum needs the grd-pull- class

<div class="grd-row">
<div class="grd-col-1-3">
<div class="grd-col-1-3 grd-pull-right">
<div class="grd-row">
<div class="grd-col-1-5">
<div class="grd-col-1-5 grd-pull-right">
<div class="grd-col-1-5">
<div class="grd-col-1-5">

Now the real beauty of flexbox - dynamic content vertical alignment!

<div class="grd-row">
<div class="grd-col-1-4 grd-align-center">
<div class="grd-col-3-4 grd-align-center">
<div class="grd-row">
<div class="grd-col-1-4 grd-align-end">
<div class="grd-col-3-4 grd-align-end">
